# Specification

## Summary
**Goal:** Enable users to export and download all project source code as a compressed archive.

**Planned changes:**
- Add a code export endpoint that bundles all frontend and backend source files
- Include configuration files and project metadata in the export
- Provide a download mechanism through the user interface
- Preserve the original directory structure in the generated archive

**User-visible outcome:** Users can click an export button to download a complete archive (zip) of all project source code, configuration files, and metadata, allowing them to work with the code locally or migrate it elsewhere.
